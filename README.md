# OptiPay Frontend Prototype

Frontend-only React and TypeScript prototype for OptiPay, built with Vite.

## Included screens

- Authentication and business onboarding
- Command center dashboard
- Vendor directory
- Individual vendor ledger
- Smart inventory manager
- Liquidity optimizer
- Universal transaction entry form

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- This version is frontend only and uses mock data.
- Vendor balances, risk cards, payment planning, and transaction actions are simulated in UI state.
- The design is mobile-friendly and can be extended with routing or a backend later.
